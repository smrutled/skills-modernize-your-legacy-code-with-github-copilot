| TC-03 | Credit the account with a valid amount. | Application is running and the current balance is 1000.00. | 1. Launch the application. 2. Select option `2` (Credit Account). 3. | TC-04 | Debit the account with an amount less than the available balance. | Application is running and the current balance has been restored to 1000.00, or use a fresh run. | 1. Enter `200.25`. | The balance decreases to `799.75` and the updated balance is displayed. |  | Pending | Confirms debit subtracts when funds are available. |
| TC-05 | Debit the account with an amount greater than the available balance. | Application is running with current balance of 1000.00 (or after a known starting balance). | 1. | TC-06 | Permit a debit equal to the available balance. | Application is running with balance equal to the debit amount. | 1. Launch the application. 2. Select option `3` (Debit Account). 3. | TC-08 | Maintain updated balance across operations within the same run. | Application is running. | 1. Launch the application. 2. Select `2` and credit `100.00`. 3. Select `1` to view balance. 4. | TC-09 | Data layer read/write behavior is reflected in displayed totals. | Application is running with default balance. | 1. Launch the application. 2.
